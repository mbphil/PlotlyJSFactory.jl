# PlotlyJSFactory

A package that extends the functionality of plotlyjs via [PlotlyJS.jl](https://github.com/sglyon/PlotlyJS.jl)

## Installation
```julia
Pkg.clone("https://github.com/mirestrepo/PlotlyJSFactory.jl.git")
```
